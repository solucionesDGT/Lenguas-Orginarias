⚙️ Paso único de configuración — API Key de Google Drive Para mostrar contadores reales (cantidad de archivos en Drive) necesitas agregar tu clave de API:

1. Abre console.cloud.google.com → Crea un proyecto nuevo.
2. Ve a APIs y servicios → Biblioteca → Busca y habilita "Google Drive API".
3. Ve a APIs y servicios → Credenciales → + Crear credencial → Clave de API.
4. Copia la clave generada y pégala en el HTML donde dice PEGA_TU_API_KEY_AQUI.
5. En "Restricciones de la clave" → Restricción de API → Google Drive API (recomendado).

ℹ️ ¿Cómo subir un documento?

  1. Selecciona tu nombre.
  2. Elige el tipo de documento.
  3. Haz clic en "Abrir carpeta en Drive".
  4. Se abre tu carpeta exacta en Drive.
  5. Arrastra o usa "+ Nuevo" para subir el archivo.

🔄 Actualizar contadores

Después de subir un archivo, regresa al panel y presiona Actualizar datos en la barra superior para ver el nuevo conteo.
